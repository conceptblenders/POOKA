# Diagrams

Editable source diagrams for the POOKA Design Paper, in draw.io format. Open and edit them with [diagrams.net](https://www.diagrams.net/) (draw.io). Export rendered images into `../figures/` for use in the paper.

## Files

- `core-concepts.drawio`: the Core Concepts and their relationships (see chapter 8, relationships per section 9.2).
- `architectural-layers.drawio`: the Knowledge, Governance and Behavior layers (see chapter 9, section 9.1).
- `prompt-vs-architecture.drawio`: contrasting runtime prompt context with persistent architecture (see chapter 2, sections 7.2 and 8.5).

These diagrams represent concepts already defined in the paper. They are not yet embedded in the paper.
